📌 Overview

This project presents an interactive Air Quality Index (AQI) dashboard designed to analyze pollution trends across different cities and states in India.

It provides meaningful insights into air quality by identifying high-risk regions, pollutant contributions, and overall environmental patterns using data visualization techniques.


🚀 Features
📊 Identify Best & Worst Cities based on AQI
🌫️ Analyze Pollutant-wise trends (CO, PM2.5, NO2, Ozone, etc.)
🗺️ Interactive geographical visualization of AQI levels
📈 Insights into Top Polluted Cities & States
🎯 Clean and modern dark-themed dashboard UI
🔍 Dynamic filters for better data exploration


📊 Key Insights
📍 Rohtak recorded an AQI of 394 (Very Poor)
📉 Significant variation in pollution levels across states
🌫️ Pollutants like PM2.5 and CO contribute heavily to AQI
🏙️ Urban areas show higher pollution concentration
